🚀 GET STARTED IN 5 MINUTES!

═══════════════════════════════════════════════════════════════════════

STEP 1: START XAMPP
═══════════════════════════════════════════════════════════════════════
  1. Open XAMPP Control Panel
  2. Click [Start] next to Apache
  3. Click [Start] next to MySQL
  4. Both should show GREEN

═══════════════════════════════════════════════════════════════════════

STEP 2: SETUP DATABASE
═══════════════════════════════════════════════════════════════════════
  1. Open browser: http://localhost/phpmyadmin
  2. Click [New] in left sidebar
  3. Database name: library-db
  4. Click [Create]
  5. Click [Import] tab
  6. Choose file: database/library-db.sql
  7. Click [Go] - wait for success message
  8. Click [Import] again
  9. Choose file: database/sample-data.sql
  10. Click [Go] - you now have 16 books!

═══════════════════════════════════════════════════════════════════════

STEP 3: OPEN APPLICATION
═══════════════════════════════════════════════════════════════════════
  Open browser: http://localhost/library_project/

  You should see the E-Library homepage! 🎉

TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════

❌ Page not loading?
   → Check XAMPP Apache is running (green)
   → Verify URL: http://localhost/library_project/ (not library-project)

❌ Database connection error?
   → Check MySQL is running in XAMPP
   → Verify database name is: library-db (with hyphen)
   → Check includes/db.php settings

❌ No books showing?
   → Import sample-data.sql file
   → Check books table in phpMyAdmin

❌ Styles broken?
   → Press Ctrl+F5 (hard refresh)
   → Check assets/css/ folder has all files

❌ Dark mode not working?
   → Check browser console (F12) for errors
   → Verify theme-init.js and dark-mode.js exist
